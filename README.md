## 使用须知
1) 该项目基于nodejs完成，启动时会占用一个TCP端口和一个UDP端口
2) 该项目支持节点分发，将连接分发到不同服务器，有效降低封号概率
3) 欢迎翻版成其他语言，本项目源文件带有大量详细的注释
4) 个人搭建请注意安全！！！
